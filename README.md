# Inference with K-means Implementation

## Requirements

- **Python**: 3.8 or higher

### Python Libraries

```
numpy>=1.21.0
scipy>=1.7.0
scikit-learn>=1.0.0
matplotlib>=3.4.0
```

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/mabarbar/inference-with-k-means-implementation.git
cd inference-with-k-means-implementation
```

### 2. Dependencies

Make sure you're in the project root directory and have installed dependencies:

```bash
pip install -r requirements.txt
```

## Running

```bash
# Run experiments (for us it was around 30 minutes)
python main.py
```
